## Flagship Project: DocFlow

### The Problem
A 50-person company was spending **$46,800/year** on document management subscriptions:
- Adobe Acrobat Pro ($25/user/mo)
- DocuSign ($40/user/mo)
- Project tracking tools ($13/user/mo)

### My Analysis
I mapped the full document workflow, interviewed stakeholders, and identified that **80% of paid features were unused**. The core needs were simple: merge, split, sign, and track documents.

### The Solution
**DocFlow** — a single desktop application that replaces all three subscriptions:
- **Merge & Split PDFs** → replaces Acrobat Pro
- **Digital Signatures** → replaces DocuSign
- **Document Workflow Log** → replaces project tracking

### The Result
**$46,800/year in savings.** Break-even: month one. Built with open-source tools. Zero recurring costs. One EXE file.
